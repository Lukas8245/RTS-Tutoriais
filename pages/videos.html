Como baixar vídeos da internet

Ferramenta

Iremos utilizar do yt-dlp, uma ferramenta open source que permite baixar vídeos de todos esses sites.

Instalação

Método recomendado

Basta colar o seguinte comando no powershell do windows:

winget install -e --id Gyan.FFmpeg;winget install -e --id yt-dlp.yt-dlp

Ou se estiver usando linux (baseado em debian ou ubuntu):

sudo apt install yt-dlp

Para baixar algum vídeo quando instalado dessa maneira, basta abrir um CMD qualquer.

Por padrão o CMD abre na pasta do usuário, digitando apenas "cd dow" e apertando tab você já estará na pasta Downloads.

Se quiser baixar para área de trabalho só digitar "cd desk" e dar tab.

Método avançado

  1. Baixe a última release do yt-dlp (se estiver no windows yt-dlp.exe)
  2. Crie uma pasta e mova o executável para ela

Somente com isso já daria para utilizar o programa, mas de maneira bem limitada já que falta o ffmpeg.

Para instalar o ffmpeg (específico do yt-dlp):

  1. Baixe a última release do ffmpeg (se estiver no windows ffmpeg-master-latest-win64-gpl.zip)
  2. Extraía para outra pasta que não seja a do yt-dlp
  3. Dentro da pasta extraída vá em bin
  4. Copie os arquivos ffmpeg.exe e ffprobe.exe para a pasta do yt-dlp
  5. Os demais arquivos não são importantes e podem ser apagados

No final sua pasta com o yt-dlp deve ter somente:

  • ffmpeg.exe
  • ffprobe.exe
  • yt-dlp.exe

Para baixar algum vídeo quando instalado dessa maneira, será necessário abrir um CMD na pasta raiz do programa.

A maneira mais fácil de fazer isso é abrir a pasta no windows explorer, clicar no endereço dela e digitar "CMD" sem as aspas.

Utilização

Existe uma quantidade enorme de opções disponíveis, a própria página do github tem um README enorme explicando.

Mas por simplicidade irei ensinar apenas dois comandos básicos que já serve para muitos casos.

Apenas abra um CMD e digite os comandos. Vai aparecer algumas informações, assim que ele terminar a mídia baixada vai estar disponível na pasta que você abriu o CMD.

Baixar um vídeo

yt-dlp {link do vídeo}

Baixar um áudio

yt-dlp -t "mp3" {link do vídeo}

Exemplo

yt-dlp -t "mp3" https://www.youtube.com/watch?v=dQw4w9WgXcQ&pp=ygUJcmljayByb2xs

Aviso playlists

Cuidado quando for copiar o link de algum vídeo que está em uma playlist porque pode ser que o programa tente baixar todos os vídeos.

Se estiver baixando do youtube basta apagar tudo que estiver depois de &list=.

Informações bônus

Preset alias

O -t na verdade é um preset alias que converte para um formato específico através do ffmpeg.

Existem alguns outros, a lista completa está aqui.

Resolução específica

yt-dlp -f "bv[height<=480]+ba" {link do vídeo}

Nesse exemplo foi utilizado a resolução 480p, mas você pode mudar para qualquer outro valor.
